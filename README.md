# seleniumbase


## get started

* install seleniumbase
``` .sh
pip install git+https://github.com/sasakidmap/seleniumbase
```


## execute test

* e.g. non basic auth
``` .sh
python test_suite_sample.py https://example.com
```

* e.g. basic auth
``` .sh
python test_suite_sample.py https://example.com -u user -p password
```
